# Apps

User-facing and operator-facing applications live here.

```text
apps/api      Public backend API for chat, search, case intake and admin operations.
apps/web      Citizen-facing web app.
apps/admin    Internal/admin dashboard for ingestion, quality and review workflows.
apps/lawyer   Advocate review workspace.
```

The current repository is database/pipeline-first. These app folders define the product boundary and are ready for implementation after the data and retrieval contracts stabilize.
